# sergiopiza.com

Personal portfolio and resume site for Sergio Piza — Data Engineer and Backend Developer.

## Stack

- **Nuxt 3** — Vue-based fullstack framework
- **Tailwind CSS** — utility-first styling
- **Prismic / SliceMachine** — CMS integration
- **GSAP + Three.js** — animations and 3D background
- **pdfmake** — resume PDF generation
- **NuxtHub** — deployment on Cloudflare

## Pages

- `/` — portfolio homepage (about, experience, wins, skills, contact)
- `/resume` — interactive resume with PDF download

## Development

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
npm run generate  # static export
npm run preview
```
